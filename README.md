# 👾 FaceHack Age & Gender Detector

This project uses OpenCV, DNN, and Caffe models to detect faces, predict gender and age — with a retro hacker-styled GUI using Tkinter.

## 🔧 Tech Stack
- Python 3.11
- OpenCV
- Tkinter
- DNN + Caffe pre-trained models
